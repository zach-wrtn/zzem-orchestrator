# Task 003: 닉네임 자동 생성

- **Group**: 2
- **AC**: 2.6

## Target

한국어 형용사 + 동물 + 4자리 숫자 포맷의 닉네임 자동 생성 API를 구현한다.

## Context

- 최초 앱 진입 시 시스템이 자동으로 닉네임을 생성해야 함
- 포맷: "빛나는고양이1234"
- 불쾌한 조합 방지를 위해 사전 검수된 단어 목록 사용

## Objective

`GET /api/v1/nickname/generate` — 랜덤 닉네임 생성 반환

## Specification

### 닉네임 포맷
- `{형용사}{동물}{4자리숫자}`
- 형용사 목록: 긍정적/중립적 형용사 (최소 20개)
- 동물 목록: 일반적 동물 (최소 20개)
- 4자리 숫자: 0000~9999 랜덤
- 불쾌한 조합이 될 수 있는 단어 제외

### 결과
- 생성된 닉네임은 2~20자 제한 내에 들어야 함
- 인증 불필요 (최초 가입 시 호출)

## Acceptance Criteria

1. `GET /api/v1/nickname/generate` 호출 시 `{ nickname: string }` 형태로 응답한다
2. 닉네임은 `{형용사}{동물}{4자리숫자}` 포맷이다
3. 닉네임 길이가 2자 이상 20자 이하이다
4. 동일 요청을 여러 번 호출하면 다른 닉네임이 반환된다 (랜덤성)
5. 형용사와 동물 목록에 불쾌한 단어가 포함되지 않는다

### Implementation Hints

- 단어 목록은 상수 파일로 관리 (`@common/constant/` 패턴 참조)
