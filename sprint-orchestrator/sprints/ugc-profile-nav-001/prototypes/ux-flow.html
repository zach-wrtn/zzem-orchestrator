1. 앱 진입 & 탭 네비게이션

🏠
탭 1: 홈
AC 2.1 | Default Landing
그리드 피드. ZZEM 로고(좌) + 크레딧·알림(우) 헤더. 3열 콘텐츠 그리드.
콘텐츠 탭 → 세로 스와이프 피드
알림 아이콘 → 알림 화면 PRD 3
프로필 영역 → 타 유저 프로필
탭 전환
🔍
탭 2: 탐색
AC 2.1
홈과 동일한 콘텐츠 풀/추천 로직. 그리드 피드 숏컷.
콘텐츠 탭 → 세로 스와이프 피드
탭 전환
👤
탭 3: MY (프로필)
AC 2.1, 2.2
프로필 헤더 + 게시물/비공개/좋아요 3탭. 디폴트 탭: 공개→비공개→게시물 순 폴백.
⚙️ → 설정 화면
프로필 편집 버튼 → 편집 화면
프로필 공유 버튼 → OS 공유 시트
콘텐츠 탭 → 세로 스와이프 (탭 콘텐츠만)
비회원 → 로그인 리다이렉트
비회원 규칙: 비회원은 홈(탭1)만 탐색 가능. MY 탭, 알림, 인터랙션 버튼 탭 시 → 로그인 페이지로 리다이렉트.

2. 프로필 상세 플로우

📝
프로필 편집
AC 2.4, 2.6
프로필 이미지 변경 + 닉네임 수정 (2~20자). 랜덤 닉네임 생성 버튼.
이미지 탭 → 카메라/앨범 바텀시트
저장 → 프로필 화면 반영
← 뒤로 → MY 프로필
📷
이미지 소스 선택
AC 2.4
카메라 촬영 / 앨범에서 선택. 바텀시트 오버레이.
카메라/앨범 → 이미지 크롭 화면
취소 → 바텀시트 닫기
📤
프로필 공유
AC 2.3
OS 기본 공유 시트. 딥링크 URL 포함 (OG Image 미포함).
공유 완료/취소 → 프로필로 복귀
닉네임 자동 생성 (AC 2.6): 최초 프로필 생성 시 시스템이 자동으로 "형용사+동물+4자리숫자" 닉네임 생성. 이후 프로필 편집에서 자유롭게 변경 가능.

3. 설정 플로우

⚙️
설정 화면
AC 2.8
7개 메뉴: 프로필 편집 / 알림 설정 / 차단 관리 / 이용약관 / 개인정보 / 로그아웃 / 탈퇴
프로필 편집 → 편집 화면
알림 설정 → "준비 중" PRD 3
차단 관리 → "준비 중" PRD 3
이용약관/개인정보 → WebView
로그아웃 → 확인 다이얼로그
탈퇴 → 탈퇴 화면
⚠️
로그아웃 확인
AC 2.8
"정말 로그아웃 하시겠어요?" 확인/취소 다이얼로그.
확인 → 로그아웃 → 홈(비회원)
취소 → 설정 화면 복귀
🚧
준비 중
PRD 3 대기
알림 설정, 차단 관리 진입 시 노출. PRD 3에서 실제 기능 연결.

4. 타 유저 프로필 플로우

👥
타 유저 프로필
AC 7.1, 7.5
프로필 헤더 + 게시물 탭만 노출. 비공개/좋아요 탭 없음. 편집 버튼 없음.
콘텐츠 탭 → 세로 스와이프 (해당 유저만)
⋯ 더보기 → 바텀시트
페르소나 계정 → 생성 화면
⋯
더보기 메뉴
AC 7.1
프로필 URL 복사. 차단/신고는 PRD 3에서 추가.
프로필 URL 복사 → 클립보드 + 토스트
차단하기 PRD 3
신고하기 PRD 3
📱
세로 스와이프 피드
AC 2.5
콘텐츠 상세 보기. 진입 맥락에 따라 해당 탭/유저의 콘텐츠만 노출.
프로필 영역 탭 → 타 유저 프로필
← 뒤로 → 이전 화면
진입점: 세로 스와이프 피드의 좌측 하단 프로필 영역 탭 / 팔로워·팔로잉 리스트에서 유저 탭 → 타 유저 프로필로 이동.

5. 생성 후 랜딩 플로우

🎨
밈 생성 (필터 기반)
AC 2.7
필터 기반 밈 생성 요청. isPublished=true로 생성.
생성 완료 → MY 프로필 → 게시물(공개) 탭
✏️
밈 생성 (커스텀 프롬프트)
AC 2.7
커스텀 프롬프트 기반 생성. isPublished=false로 생성.
생성 완료 → MY 프로필 → 비공개 탭
⏳
생성 중/실패 상태
AC 2.7
생성 중 스피너, 실패 시 재시도 버튼. 해당 탭에서 인라인으로 노출.
규칙: 이 랜딩 규칙은 생성 직후 진입에만 적용. 일반 진입(MY 탭 직접 탭 등)은 AC 2.1 디폴트 탭 규칙을 따름.

6. 기존 데이터 마이그레이션

💾
기존 밈 → 비공개 탭
비즈니스 룰
기존 생성 밈은 isPublished 필드 미존재 = 비공개로 간주. 스키마 default만, No Batch 마이그레이션.
