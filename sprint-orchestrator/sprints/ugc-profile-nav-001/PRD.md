## 스코프 요약

앱의 네비게이션 골격(하단 탭 바 3탭)과 프로필 시스템을 구축한다.

### 핵심 기능

1. **3탭 네비게이션**: 홈(그리드 피드) / 탐색(돋보기) / 프로필(MY)
2. **프로필 시스템**: 프로필 이미지, 닉네임, 팔로워/팔로잉/재생성된 카운트
3. **MY 3탭**: 게시물(공개) / 비공개 / 좋아요
4. **프로필 편집**: 이미지(카메라/앨범), 닉네임(2~20자)
5. **프로필 공유**: OS 공유 시트 + 딥링크
6. **닉네임 자동 생성**: 형용사 + 동물 + 4자리 숫자
7. **타 유저 프로필**: 게시물 탭만 노출, 더보기 메뉴
8. **설정 화면**: 프로필 편집, 알림 설정(준비중), 차단 관리(준비중), 약관, 개인정보, 로그아웃, 탈퇴
9. **생성 후 랜딩**: 필터→게시물탭, 프롬프트→비공개탭
10. **기존 밈 마이그레이션**: isPublished=false 기본값
