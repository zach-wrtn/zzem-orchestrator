## ⚠️ Fix Ready for QA — free-filter-qa-fix / group-5

**Root Cause**
추천 탭(grid feed exp group B/C) 응답을 만드는 `readGridFeedRecommendations` 가 `rosterContext` 를 로드하지 않고, `enrichRecommendedFilters` 가 `isFreeFilter:false` / `hasGeneratedToday:false` 로 하드코딩. 같은 모듈의 `enrichFilters` (일반 탭/무료 탭) 와 `readOne` (단건/프리뷰) 은 rosterSlot 기반 동적 판정 적용. free-tab-diversification 의 US-7 (외부 진입점 파리티) 가 단건/deeplink 만 커버하고 추천 탭 리스트 응답 path 의 enrichment gap 이 잔존.

**Fix Summary**
3 위치 보강 (BE 단일 모듈):
1. `readListByGridFeed` 시그니처에 `appVersion` 추가, 내부 `readListV2` fallback 호출 모두에 전달 (rosterContext capable check 정확성)
2. `readGridFeedRecommendations` 가 `loadRosterContextIfCapable(capable, userId)` 로 rosterContext 로드 후 `enrichRecommendedFilters` 에 전달
3. `enrichRecommendedFilters` 가 `enrichFilters` 와 동일한 rosterSlot 분기 (isFreeFilter / hasGeneratedToday / themeTag 동적 판정)
Controller 의 `readListByGridFeed` 호출에 appVersion 추가.
