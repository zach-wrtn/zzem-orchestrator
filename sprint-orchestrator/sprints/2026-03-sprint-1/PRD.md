# PRD Reference

이 스프린트의 PRD 원본: [PRD-001-ugc-platform](../../../docs/prds/PRD-001-ugc-platform.md)

## 스프린트 스코프 요약

- US1: 피드 공개 (공개/비공개 토글, 페이백 바텀시트, CTA 분기, 커스텀 프롬프트 제한)
- US2: 프로필 (프로필 화면, 탭 구조, 프로필 공유, 프로필 편집)
- US3: 좋아요 (셀프 좋아요, 좋아요 탭, 추천 시그널)
- US4: 크레딧 페이백 (페이백 조건, 크레딧 히스토리, 최초 진입 UX)
- US5: 알림 & 푸시 (카테고리별 알림, 팔로우 알림, 알림센터)
- US6: 팔로우 (팔로우 상태, 팔로우 리스트, 추천 부스트)
- US7: 타 유저 프로필 및 사회적 기능 (차단, 신고, 의견 보내기, 페르소나)
