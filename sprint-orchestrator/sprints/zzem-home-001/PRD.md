## Scope Summary

ZZEM 앱의 홈 화면 구현. 사용자가 AI 기반 콘텐츠 생성 템플릿을 탐색·발견하고 빠르게 제작에 진입할 수 있는 허브 역할.

### Core Features
1. **헤더**: 로고, 코인 버튼, 알림 벨 (뱃지 포함)
2. **퀵 액션 메뉴** (4종): 비디오 생성, 이미지 생성, 댄스 챌린지, 주인공 바꾸기 (Beta)
3. **신규 템플릿**: 가로 스크롤 카드 리스트 + 전체보기
4. **실시간 랭킹**: Top 3 + 더보기 + 만들기 버튼
5. **카테고리 필터 칩**: 1차(추천/무료/비디오/이미지/조합/말하는아기) + 2차(연인/반려동물 등)
6. **2열 매거진 콘텐츠 그리드**: 썸네일, 템플릿명, 크리에이터, 좋아요
7. **하단 네비게이션** (3탭): 홈, 검색, My + 알림 뱃지

### User Stories (7)
- US1: 홈 화면 진입 및 콘텐츠 탐색
- US2: 퀵 액션을 통한 콘텐츠 생성 진입
- US3: 신규 템플릿 탐색
- US4: 실시간 랭킹 확인 및 인기 템플릿 제작
- US5: 카테고리 필터를 통한 콘텐츠 탐색
- US6: 콘텐츠 그리드 탐색 및 상호작용
- US7: 하단 네비게이션을 통한 화면 전환

### Boundary
- **Must**: 퀵 액션, 신규 템플릿, 실시간 랭킹, 필터 칩, 매거진 그리드, 하단 네비게이션, 헤더
- **Should**: 다크모드, 스켈레톤 로딩, 풀다운 새로고침
- **Won't**: 데스크톱/태블릿 반응형, 비로그인 사용자 홈
