      iOS Settings / Android Settings / 카카오톡 설정 등 표준 패턴.

      - "section header 추가 — '내 정보' / '안전' / '도움' 등 라벨 — 카피 작성 부담"
    would_break_if: "Figma 가 단일 list 인 경우 — section gap 만 제거하면 됨"

    rationale: |
      Figma 원본 아이콘 미접근 — lucide-style 표준 아이콘으로 매핑.
      모든 아이콘 same stroke-width(2) + same size(24x24) — 시각 일관성 (AC-4.4).

      - "icon 생략 — 텍스트만 (단순)"
      - "Material/Heroicons 등 다른 icon set"
    would_break_if: "Figma 가 brand custom icon 사용 — 시각 톤 차이"
