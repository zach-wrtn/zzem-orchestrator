      form persona 강제 룰 #3 (error message 입력 직하) 충족 위해 RowError 가 primary.
      Toast 는 보조 — 사용자 시선이 row 부근에 있을 때 RowError 만으로 충분하나,

      Figma MCP 접근 권한 차단(거절됨) — Figma 원본 카피 추출 불가.
      PRD 도 카피 명시 없음. ZZEM 다른 settings 화면의 카피 톤을 참고하여 placeholder.
