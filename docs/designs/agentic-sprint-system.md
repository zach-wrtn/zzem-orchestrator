## Vision

### 10x Check
PRD 하나를 넣으면 MemeApp + meme-api의 코드가 자동으로 생성되고, 테스트되고, PR까지 만들어지는 완전 자동 스프린트 시스템. 개발자는 PRD 작성과 최종 리뷰만 담당.

## Approach Decision
**Approach B (Lightweight Orchestrator)** 선택 — Full Orchestrator(A) 대신 최소 인프라로 핵심 가치를 먼저 검증하고 확장. LLM 태스크 분해 품질이 미검증 상태에서 과도한 인프라 투자를 피함.

### 대안 비교
| Approach | Summary | Effort | Risk |
|---|---|---|---|
| A) Full Orchestrator | 별도 디렉토리 + 10개이상 셸 스크립트 | L (CC: ~4h) | Medium |
| **B) Lightweight** | 기존 CLAUDE.md 확장 + 단일 실행 스크립트 | **S (CC: ~1h)** | **Low** |
| C) Claude Code Agent | Agent tool로 서브에이전트 스폰 | M (CC: ~2h) | Medium |
