Do's and Don'ts

Do
  • Theme key로 색상 참조 (theme["zzem_violet_500"])
  • Typo 컴포넌트로 텍스트 (Typo.Subtitle4)
  • useShadow() 훅으로 그림자 통일
  • 선택 상태 = violet_500/600
  • 비활성 = opacity 0.4
  • gap: 8px (기본), 4px (icon-text)
Don't
  • Hex 하드코딩 (#8752FA 직접 사용)
  • Inline fontSize/fontWeight 지정
  • 커스텀 shadow variant 생성
  • 카드에 pill radius(9999) 사용
  • Pure black(#000) 텍스트 사용
  • Elevation xl을 카드에 사용
