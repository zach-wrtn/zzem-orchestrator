## MY 프로필과의 차이

| 요소 | MY | Other User |
|------|-----|------------|
| 프로필 편집 버튼 | O | X |
| 프로필 공유 버튼 | O | X |
| 팔로우 버튼 | - | X (PRD 3 스코프 아님) |
| 탭 | 게시물·비공개·좋아요 | 게시물 only |
| 헤더 | 아이디 + 설정 아이콘 | 닉네임 + 인증 뱃지 + ⋯ 더보기 |

## 공통 유지 요소

- 100px 원형 프로필 이미지
- 3열 카운트 (팔로워·팔로잉·재생성된)
- 피드 그리드는 카운트 바로 아래에서 시작 (바이오 텍스트 없음)

## 헤더

닉네임 + 인증 뱃지 (중앙) / `⋯` 더보기 아이콘 (우측).
