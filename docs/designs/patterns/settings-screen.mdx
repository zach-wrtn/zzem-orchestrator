## Layout

텍스트 행 리스트를 섹션별로 그룹화. 섹션 간에는 12px 높이 회색 배경 디바이더로 구분.

## 행 규격

- **아이콘 없음** — 텍스트만 + 선택적 chevron (`>`)
- `pl-20 pr-16 py-12`, Pretendard Medium 16px
- 디바이더: 12px 높이, bg `#f7f7f7`

## 섹션 구조

1. **계정** — 계정 정보(이메일), 비밀번호
2. **설정** — 알림 설정, 차단 관리
3. **정보** — 이용약관, 개인정보, 고객센터
4. **계정 관리** — 탈퇴하기, 앱 버전

## 로그아웃 버튼

**하단 고정 풀폭 버튼** (리스트 아이템이 아님). 상세는 [Button](/zzem-orchestrator/system/components/button/) 참조.

- bg `#f1f1f1`, rounded-16, h-56
- Pretendard SemiBold 18px `#656565`
